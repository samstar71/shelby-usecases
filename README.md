# Shelby Use Case Explorer

This repository explores how Shelby can support real-world applications that require high-performance decentralized storage.

## Potential Use Cases

### AI Model Training
Large datasets require fast and reliable storage infrastructure.

### Video Streaming
Content delivery networks can be decentralized using Shelby.

### Gaming
Game assets and updates require high throughput storage.

### Scientific Research
Large biological and genomic datasets could be stored and distributed globally.

### DePIN Data Networks
IoT and sensor networks generate large volumes of real-time data.

## Why This Matters

Shelby combines decentralized storage with high-performance networking using the DoubleZero infrastructure and verification through Aptos.

This creates a foundation for a new generation of decentralized data infrastructure.
